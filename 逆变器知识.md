# 逆变器

## 1.基本知识

### 1.1 硬件电路H桥

* H桥电路-4开关构成

  ~~~text
  Vin+     
  ---------|
           +-----------------+
           |                 |
          Q1                 Q3
           |                 |
           o--------LR-------o
           |                 |
          Q2                 Q4
           |                 |
           +-----------------+
  ---------|
  Vin-   
  ~~~

  * 控制方式

    Q1和Q4是一组,当Q1和Q4开通时,电流走向VIN+->Q1->R->Q4->Vin-

    Q2和Q3是一组,当Q2和Q3开通时,电流走向VIN+->Q3->R->Q2->Vin-

    以上是带纯阻性负载的情况,电流和电压相位不会变,如果带负载是感性,那还是需要考虑其他的.

    通过这种H桥加开关管的控制方式就可以把DC->AC

  * PWM占空比调制

    Ts-整个周期,Ton-开关导通时间,Toff-开关关闭时间,占空比D=Ton/Ts
  
    ~~~text
    Ton	    Toff
    -------		   ---------Ts
    |	   |_ _ _ _|	    |
    ~~~
    
    Urms = D*Udc,可以看出D和有效值是具有相关性的,Udc一般是定值,因为是电源控制所以前端输入必须要比较稳定.
    
    所以要想办法把占空比调制为SIN波形,假设做平均值量化.我们需要取出有效值,Urms=D*Udc
  



## 2.调制方式

简单补充：交流电形式u(t) = Um * sin(ωt + φ)

-   **Um**：峰值（最大值），对于220V有效值的交流电，Um = 220 × √2 ≈ 311V
-   **ω**：角频率，ω = 2πf-f=50hz
-   **φ**：初相位
-   u(t) = √2 * U_rms * sin(ωt + φ)

调制比：M这个参数理论最大值为1，过了1就是过调制区会失真。

### 2.2 双极性调制

* 顾名思义双极性调制,在一个周期内.有正值也有负值,这个时候的损耗比较大,因为他的电压切换是一瞬间,在UAB+--->UAB-,这是双极性调制的一个缺点.但是也可以使用.输出UAB+的时候是在,Q1和Q4导通,输出UAB-是Q2和Q3导通.开关管的频率是始终一以高频的方式进行导通.

  运算方式:调制波+1*(1/2)=这样做的目的是为了把调制波的幅值映射到0-1之间从而来比较载波和调制波大小,而控制pwm来进行输出.

### 2.3 单极性调制

#### 2.3.1 单极性调试方式一

* 单极性调制是,在sin的正半轴的时候,输出V+->Q1->Q4->V-,输出正的UAB+.
* 输出0=>-Q1管子关闭->Q2打开,Q4保持开,此时输出0电压,电流只在Q2->Q4之间续流,此时2个管子都是0电位,所以能输出0V,在这个正半轴的状态下,Q1和Q2是高频切换管,Q3和Q4是低频管(50HZ),正半轴的情况下只开Q4,Q3不开.
* 负半周的情况下,我们改变高频管和低频管,Q1和Q2在负半轴做低频管,Q3和Q4做高频管,因为要输出-UAB和0那么Q1是不允许导通的,Q2一直导通.
* 输出-UAB=>V+->Q3->Q2->V-,因为规定了参考方向,所以输出负值,输出负电压(UAB)
* 输出0电压=>关闭Q3->打开Q4此时的方向是在Q2和Q4之间续流,两点电位相同故输出0电压.以此来循环.就能得到一个比较好的正弦信号.这种方式主要考虑的是,开关的均衡磨损.

#### 2.3.2 单极性调试方式二

* 这种方式主要是固定高频管和低频管
* Q1和Q2始终当高频管,Q3Q4始终当低频管
* 在sin的正半轴时刻,我们Q1和Q4导通,输出UAB+=>此时电流放向->v+->Q1->rL->Q4->v-,输出的是正电压UAB
* 输出0电压->关闭Q1打开Q2->Q4始终导通->电流回路如下->Q2->Q4->因为他们都是共-级没有压差,所以输出0电位,输出电压就是0
*  sin的负半轴时刻,我们控制Q3导通,Q4关闭.此时输出-Uab->方式=>v+->Q3->Q2->v-,输出UAB-
* 输出0v-我们的Q3管保持导通,->Q2管关闭->Q1管导通
* 此时的电流回路->V+->Q1->Q3这两个管子进行续流,他们之间没有压差,所以输出0v电压.
* 通过以上的简述初步达成了想要的控制效果.
