### 2-2-2 查表插值的应用

* 基本思想

  * 按照上表来做解析,假设温度落在10-25摄氏度之间,放电倍率落在1-2之间,那么就可以取出4个容量数据表,2,3,2,3四个数据,综合起来就可以取4个容量值.分别是10摄氏度1c和2c,25摄氏度1c和2c.

  * 归一化插值的系数-计算公式:(当前值-左边界) / (右边界-左边界)

  * 假设当前温度是15,落在表的10到25摄氏度的区间,那么归一化后的温度系数就是:(15-10) /(25-10) = 5/15 = 0.3333,这个物理意义就是反应15摄氏度落在10到25区间内的33.3%的位置附近

  * 假设当前放电倍率为1.2倍,按照表格落在1到2的区间,按照上述的计算系数:1.2-1   / 2 -1 = 0.2 

  * 得到系数后开始进行加权计算四个边界点的贡献值value = (1-0.333)x(1-0.2)xQ11 + (1-0.333)x(0.2)Q12 + (0.333)x(1-0.2)Q13+(0.333)x(0.2)Q13= 2498mah

  * 一维插值计算公式
    y0 + (y1 - y0) * (x - x0) / (x1 - x0);

  * 解释和示例计算-温度和倍率同上

    容量值:2500 / 2000 / 2800 / 2300 

    固定放电倍率插值-插值

    所求低温容量值 = (2000 -2500) / (2-1) * (1.2-1)+2500 = -500*0.2+2500 = 2400 mah

    所求高温容量值 = (2300 - 2800) / (2-1) * (2-1)+2800 = -500 *0.2 +2800 = 2700mah


    温度插值  =  (所求高温容量值 - 所求低温容量值)/(25-10) * (15-10)+2400 = 300/15*5+2400 = 2600mah

    ## 2-3 一阶低通滤波的方法以及基本公式

### 2-3-1 理论和应用

* 一阶低通滤波-公式（离散形式）

  y[k] = α · x[k] + (1 - α) · y[k-1]

  y[k] 当前滤波输出值

  α -滤波系数（0~1）这个系数越大-响应越快，越小越平滑，是根据频率特性响应来的

  f = α /2paiT  α 处于分子项 ，所以 a越大 f越小那么高频信号滤出的越多，反之则是低频率滤出的越多

  x[k]当前时刻输入数据

  y[k-1]上一时刻滤波的输出

* 对应到SOC的显示上我们可以做出一阶低通滤波

* ~~~C
  soc_show0 = 0.9 * soc_show0 + 0.1 * (soc_show0 + adjustment);  // 平滑调整
  ~~~

##  soc的计算公式

​	soc计算公式繁多,最能直观反馈soc的公式是:SOC = 当前可用容量/电池实际容量,这个是定义式,可以根据当前的容量和总容量来反应此时还剩余多少容量.
​	示例:
​	当前可用容量5000, 实际电池容量为10000
​	soc = 5000/10000 = 0.5,此时的soc是50%,这是简单的示例

安时积分法的计算公式：SOC=SOC1+（∫idt/c)
通过上面的公式，可以看出我们要求的deta-c，就是要求容量的变化。通过时间对电流的积分来累积，初始的soc我们可以用soc-ocv的关系对照表来查表获取，但是由于积分量的存在，会一直溢出所有需要加修正的逻辑
